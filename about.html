How To Use This Page

Pick from a list of songs. Play, repeat, shuffle click next, previous ,etc.

You can use keyboard shortcuts as well

  • spacebard == play / pause
  • right arrow = next song
  • left arrow == previous song
  • 's' == shuffle
  • 'r' == repeat song

Pick up where you left off

Click the name of the song to find it in teh list (given that it is visible)

If you ever want to refresh up on how to use this site, just look for the question mark icon
